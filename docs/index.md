## Project Overview

This project demonstrates exploratory data analysis (EDA) skills using:

- GitHub for version control
- Jupyter notebooks for interactive analysis
- pandas for data manipulation
- Seaborn and Matplotlib for visualization
- Statistical analysis techniques

## Purpose

Create a custom exploratory data analysis (EDA) project using GitHub, Jupyter, pandas, Seaborn, and other popular data analytics tools to analyze U.S. state-level car crash statistics.
